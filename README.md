# How to use
```
ts-node ./src/index.ts <- Args go here

First arg: Going date
Second arg: Leaving Date
Third arg: Email to send updates to
Fouth arg: Price of how much you want to pay (could be a bit more, or a bit less)
Five arg: Where you're coming from
Sixith arg: Where you're going to

Example:
ts-node ./src/index.ts "16th March 2023" "20th March 2023" "ykosiner@gmail.com" "69420" "Brazil" "London - Heathrow"
```
